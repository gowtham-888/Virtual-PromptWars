# Virtual PromptWars — Copa Copilot

Copa Copilot is a premium, AI-assisted stadium operations and fan experience prototype designed for a FIFA World Cup 2026-style event. It combines an interactive fan companion, a live command center, a volunteer assistant, and AI-powered stadium tools in a single polished experience.

## What the project demonstrates
- Real-time crowd and operations simulation
- Multilingual fan assistance
- Volunteer support workflows
- Accessibility-focused route planning
- Secure, accessible, testable front-end behavior

## Run locally
1. Open [index.html](index.html) in a browser, or serve the folder with any static web server.
2. Run the regression tests with:
   - `npm test`

## Quality and validation
The project includes automated regression checks for accessibility, structure, and quality-focused metadata.
